# 校校系统前端界面

基于调研报告设计的校校系统前端界面，采用现代商业风格。

## 技术栈

- **框架**: React 18 + TypeScript
- **构建工具**: Vite 8
- **UI 组件库**: Ant Design 5.x
- **路由**: React Router v6
- **样式**: 自定义 CSS 变量 + Ant Design 主题配置

## 视觉效果

### 配色方案
- **主色调**: 活力蓝 (#3b82f6) - 专业、可信赖
- **渐变主题**: 蓝紫渐变 (#667eea → #764ba2) - 年轻、有活力
- **辅助色**: 青色 (#06b6d4)、绿色 (#10b981)、橙色 (#f97316)、粉色 (#ec4899)

### 背景色彩
#### 深色模式
- **主背景**: #0f172a (深蓝灰)
- **卡片背景**: #1e293b (石板灰)
- **悬浮背景**: #334155 (浅石板灰)

#### 浅色模式
- **主背景**: #f1f5f9 (浅灰蓝)
- **卡片背景**: #ffffff (纯白)
- **悬浮背景**: #f8fafc (极浅灰)

### 设计特点
- **渐变效果**: 蓝紫渐变、冷暖渐变、多彩渐变
- **玻璃态**: 磨砂玻璃效果，现代感强
- **阴影系统**: 多层次柔和阴影，立体感强
- **圆角设计**: 12-24px 圆角，亲和力好
- **流畅动画**: 弹性缓动曲线，自然舒适
- **深浅切换**: 支持一键切换深色/浅色模式，本地存储偏好

## 已实现页面

### 1. 布局框架
- 左侧固定导航栏（260px 宽，11 个菜单项）
- 顶部导航栏（70px 高，搜索、通知、校币余额、用户菜单）
- 响应式内容区域
- 背景渐变装饰效果

### 2. 首页 (Home)
- 欢迎区域（渐变背景，装饰光晕）
- 待办事项（异常卡片，红色渐变顶边）
- 数据看板（三卡片布局，渐变图标）
- 黑板报（通知、公告、活动）
- 新功能提示（NEW 标签）

### 3. 学员模块 (Students)
- 统计卡片（学员总数、在读、已结课）
- 搜索和筛选（全部/在读/已结课标签切换）
- 学员列表表格（头像、姓名、班级、状态、操作）
- 添加学员按钮（渐变背景）
- 精致分页样式

### 4. 作业模块 (Homework)
- 统计卡片（全部作业、模考、打卡、待批改）
- 作业类型标签切换（胶囊样式）
- 多维度筛选（科目、班级、状态）
- 作业列表（完成情况进度条、批改情况）
- 操作按钮（详情、预览、AI 分析、催交）

### 5. 占位页面
- 班级、课表、题库、教研、云盘、申请、待办、创意、个人中心
- 统一风格的占位界面

## 设计风格详解

### 1. 商业化
- 专业的蓝色主题，传递信任感
- 清晰的层级结构，信息组织有序
- 统一的视觉语言，品牌感强

### 2. 年轻化
- 蓝紫渐变，活力十足
- 圆角卡片，亲和力好
- 流畅动画，交互有趣
- 彩色图标，视觉丰富

### 3. 现代化
- 玻璃态效果，科技感强
- 柔和阴影，立体感好
- 渐变边框，精致细节
- 弹性缓动，自然舒适

## CSS 变量系统

主题使用 CSS 变量定义，易于定制：

```css
--primary: #3b82f6;          /* 主色调 */
--gradient-start: #667eea;   /* 渐变起始 */
--gradient-end: #764ba2;     /* 渐变结束 */
--bg-dark: #0f172a;          /* 深色背景 */
--bg-card: #1e293b;          /* 卡片背景 */
--radius-lg: 16px;           /* 大圆角 */
--shadow-primary: 0 4px 14px rgba(59, 130, 246, 0.4);
```
